## How to Use the Barcode Scanner Correctly

For best scanning results, observe the following guidance:

1. **Hold the scanner at an angle** relative to the barcode to achieve the best reading effect.
2. **Do not hold the scanner at exactly 90°** to the barcode — barcodes cannot be read normally at a perfect right angle.
3. **Align the scan line** with the barcode and ensure the entire barcode is covered by the scanner's window for decoding. Do not partially scan a barcode, as this may cause errors.

### Correct Scanning Methods (illustrated in manual):
- Hold scanner at a slight tilt, scan line aligned fully across the barcode.

### Incorrect Scanning Methods (illustrated in manual):
- Holding scanner perpendicular (90°) to barcode.
- Scanning only part of the barcode.

## Function Settings

### Operating Modes

| Mode | Function |
|------|----------|
| **Common Mode** (Instant Data Upload) | Scanned barcodes are uploaded to the computer instantly. *(Default mode.)* |
| **Inventory Mode** (Storage Mode) | Scanned barcodes are stored in internal memory. When internal memory is full, the buzzer beeps as a prompt. |
| **Data Non-Loss Mode** (Instant Upload + Storage) | Barcodes are uploaded to the computer automatically when wireless signal is present. When signal is weak or absent, barcodes are stored in internal memory and uploaded automatically once the connection is restored. |
| **Data Upload** | Uploads all data stored in internal memory to the computer, one by one. Applicable to Inventory Mode and Data Non-Loss Mode. Ensure proper wireless connection during upload; buzzer beeps twice on success and three times on failure. |
| **Display Total Number of Scanned Barcodes** | Displays the total number of stored items. Applicable to Inventory Mode. |
| **Clear Data** *(Use with caution)* | Clears all data from internal storage. |

### Mode Instructions (Details)

1. **Common Mode:** Scan the barcode to enter Common Mode. Scanned barcodes are uploaded to the computer instantly. *(Default mode.)*

2. **Inventory Mode:** Scan the barcode to enter Inventory Mode. Scanned barcodes are stored in internal memory. When the internal memory is full, the buzzer will "beep" as a prompt.

3. **Data Non-Loss Mode:** Barcodes are uploaded automatically when wireless signal is available. When signal is weak or absent, data is stored internally, and uploaded automatically once wireless connection is restored.

4. **Data Upload:** Uploads internally stored data to the computer one by one. Ensure proper wireless connection during upload. The buzzer beeps **twice** on successful upload, and **three times** on failure.

5. **Display Total Number of Scanned Barcodes:** Shows the total number of stored records. Applicable to Inventory Mode only.

6. **Clear Data:** Clears all data from internal storage.

## Pairing Settings

### 1. Pairing Barcode

Scan the **"One-Click Pairing"** barcode to initiate pairing.

### 2. Pairing Method

**One-Click Pairing Procedure:**

1. Use the scanner to scan the **"One-Click Pairing"** barcode.
2. Insert the receiver into the USB port of the computer.
3. The buzzer will stop beeping when pairing is completed successfully.

> **Note:** Pairing must be completed within **30 seconds**.

### 3. Barcode Scanner ID Setting

You can assign a custom ID to each barcode scanner using the following barcodes:

| Barcode | Function |
|---------|----------|
| **Enter ID Settings** | Activates ID configuration mode |
| **Display ID** | Shows the current scanner ID |
| **Hide ID** | Hides the scanner ID |

**Setting Method:**
- Use the wireless barcode scanner to scan **"Enter ID Settings"**, then scan the desired parameter value barcode.
- Example: To set scanner ID to **"01"**, scan **Enter ID Settings** → scan **"0"** → scan **"1"**.

**Show/Hide ID:**
- Scan **"Display ID"** or **"Hide ID"** barcode to toggle visibility of the scanner's ID.

### 4. Parameter Value Barcodes

The following barcodes represent numeric and alphanumeric values used in parameter configuration:

| Value | Barcode Available |
|-------|------------------|
| 0 – 9 | Yes |
| A, B | Yes |

## Troubleshooting

| No. | Problem | Possible Cause | Solution |
|-----|---------|----------------|---------|
| 1 | Cannot start up, or starts and shuts down immediately | Low battery | Charge in time to ensure sufficient power. The red LED indicator will be on during charging; it turns off when fully charged. |
| 2 | Barcode data is not uploaded | Scanner is not paired with receiver | Ensure the wireless barcode scanner and receiver are paired. If not, re-pair. |
| | | Barcode scanner is too far from the receiver | Check whether the scanner has a signal. If the buzzer beeps 3 times during data upload, the signal is insufficient — move closer to the receiver or switch to Inventory Mode. |
| 3 | Laser comes on, but symbol does not decode | Scanner is not programmed for this barcode type | Scan the setup barcode to enable this barcode type. |
| | | Barcode is defaced | Try scanning a test barcode of the same type. |
| | | Distance between scanner and barcode is incorrect | Move the scanner closer to or farther from the barcode. |
| | | Distance between scanner and barcode is not enough, or scan line is not crossing the full code | Make sure the scan line aims at the barcode correctly. |
| 4 | Scanned data is incorrectly displayed on the host | Incorrect input method | Change the input method to Standard English. |
| | | Other scanner settings have been changed | Restore scanner to factory defaults or contact the manufacturer service center. |
| 5 | Data is displayed as a messy code or error | Keyboard country is improperly configured | Select the keyboard layout of your country. |
| 6 | Other decoding failures | Connection errors | Disconnect power supply, reconnect host cable, reconnect power supply, restore factory defaults, and re-scan the barcode. |

## Simple Maintenance Methods

### Cleaning the Scan Window

1. Stains and dust on the scanning window can sometimes affect scanner performance. Use a **good-quality tissue** to gently wipe the scanning window, then blow any excess dust using a **dust blower**.
   - **Warning:** Using poor-quality tissue for a long time will damage the surface smoothness and affect reading accuracy.

2. The shell of the scanner may be cleaned with a **soft cloth**. If needed, clean the exterior using a soft cloth with a **mild detergent-water solution**.

2. **Do not** directly touch the laser chip or use a conductive object (such as metal) to directly observe the laser — avoid direct eye exposure.
